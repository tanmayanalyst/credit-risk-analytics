# 📌 Project Overview

Financial institutions need to identify customers who are likely to become delinquent before missed payments begin affecting business performance.

This project demonstrates an end-to-end analytics workflow, starting with Exploratory Data Analysis (EDA), followed by predictive model planning and finally the design of an AI-powered collections strategy. The proposed solution focuses on improving operational efficiency while maintaining transparency, fairness, and regulatory compliance.

---

# 🎯 Project Objectives

- Perform Exploratory Data Analysis (EDA)
- Assess dataset quality and missing values
- Identify key delinquency risk factors
- Select an appropriate predictive model
- Design a transparent AI-powered collections strategy
- Apply Responsible AI principles
- Translate analytical findings into business recommendations

# 🔍 Exploratory Data Analysis

The EDA focused on evaluating data quality before any predictive modelling.

Key activities included:

- Missing value detection
- Data quality assessment
- Correlation analysis
- Risk indicator identification
- Feature selection
- Business insight generation

---

# 🚨 Key Risk Indicators

The analysis identified the following major indicators associated with customer delinquency:

- Early missed payment behaviour (Month_1)
- High credit utilization
- Lower customer income
- Higher debt-to-income ratio
- Lower credit score

These variables were selected as the most informative inputs for the predictive model.

---

# 🤖 Predictive Model

### Selected Model

**Logistic Regression**

### Why Logistic Regression?

- Suitable for binary classification
- Produces probability-based risk scores
- Highly interpretable
- Easy to explain to business stakeholders
- Supports regulatory compliance in financial services

# 🧠 AI-Powered Collections Strategy

The proposed AI system consists of four core components:

### 1. Data Pipeline

Collects customer payment history, income, credit utilization, and account information.

### 2. Decision Engine

Evaluates customer risk using predictive analytics and business rules.

### 3. Action Layer

Automatically recommends personalized collection actions such as:

- SMS reminders
- Email notifications
- Payment plans
- Human escalation

### 4. Learning Loop

Continuously monitors customer responses and repayment behaviour to improve future decision-making.
